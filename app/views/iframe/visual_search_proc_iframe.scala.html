ขั้นตอนการทดลอง Visual Search

  1. ระบบจะกำหนดลักษณะรูปเลขาคณิต เพื่อให่ท่านทำการค้นหา (เช่น วงกลมสีฟ้า สี่เหลี่ยมสีแดง วงกลมสีเขียว)
  2. โจทย์จะปรากฏเป็นรูปเลขาคณิตซึ่งมีลักษณะรูปร่าง สี และจำนวนแตกต่างกัน
  3. ให้ท่านมองหาและคลิกรูปเลขาคณิตซึ่งปรากฏในโจทย์ ตามที่ระบบกำหนดให้ค้นหา

    4. ผลการทดลองที่ได้ไม่ได้ระบุถึงความสามารถหรือIQแต่แสดงให้เห็นว่าการประมวลสารสนเทศของมนุษย์สามารถได้รับอิทธิพลจากสิ่งเร้าหลายรูปแบบ
